# Book Tracker

**Автор:** [Торгонин Михаил]

**Дата создания:** [25.04.2026]

## Описание

Приложение для учета прочитанных книг. Позволяет добавлять книги, фильтровать их и сохранять данные в JSON.

## Возможности

* Добавление книг (название, автор, жанр, страницы)
* Фильтрация:

  * по жанру
  * по количеству страниц
* Сохранение и загрузка данных (JSON)
* Проверка корректности ввода

## Требования

* Python 3.7 или выше
* Библиотеки:

  * tkinter (обычно уже входит в стандартную поставку Python)
  * json (встроенная)

## Запуск программы

1. Сохраните файл программы с расширением `.py` (например, `main.py`)

2. Откройте терминал / командную строку и перейдите в папку с файлом

3. Запустите программу командой:
   python main.py

4. После запуска:

   * Введите данные книги
   * Нажмите "Добавить книгу"
   * Используйте фильтры
   * Сохраняйте и загружайте данные через JSON

## Пример использования

1. Введите:
   Название: 1984
   Автор: Orwell
   Жанр: Антиутопия
   Страницы: 328

2. Нажмите "Добавить книгу"

3. Используйте фильтры:

   * Введите жанр и нажмите "Применить фильтр"
   * Фильтруйте книги по количеству страниц

## Дополнительно

Данные можно сохранить в JSON файл и позже загрузить обратно в приложение.
